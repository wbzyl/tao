Jekyll-Blog HOWTO

Markdown

Posty wpisujemy używając notacji Markdown. Notacja jest opisana przez autora Johna Grubera w witrynie Daring Fireball.

Na stronie Markdownr można wpisywać tekst w notacji Markdown w jednym okienku, mając w drugim — podgląd na skonwertowany na HTMl tekst.

Kolorowanie kodu

Kod wpisujemy tak:

{% highlight ruby %}
def foo
  puts 'foo'
end
{% endhighlight %}

A tak to wygląda po podkolorowaniu:

def foo
  puts 'foo'
end

Kod jest kolorowany przez narzędzie pygmentize. Wykonanie polecenia:

pygmentize -L lexers
...
* css+erb, css+ruby:
* erb:
* js+erb, javascript+erb, js+ruby, javascript+ruby:
* rhtml, html+erb, html+ruby:
* xml+erb, xml+ruby
...

wypisuje listę obsługiwanych języków programowania.
